# 📖 О проекте

Добро пожаловать в проект **JP-Trip**.

Это не просто план путешествия и не список достопримечательностей.

Это персональный Travel Book, который создается специально для нашей семьи и будет сопровождать нас на протяжении всей поездки в Японию.

Главная цель проекта — подготовить максимально удобный, красивый и практичный путеводитель, который поможет путешествовать без лишнего стресса и позволит получить максимум впечатлений.

# 🎯 Концепция путешествия

Мы не стремимся увидеть всё.

Мы хотим увидеть лучшее.

Основные принципы поездки:

- один основной район в день;
- минимум лишних переездов;
- комфорт ребенка важнее расписания;
- качественная японская кухня;
- разумный шопинг;
- время для отдыха каждый день.

Это не универсальный путеводитель.
